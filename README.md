# 徐州美食地图

一个正在开发中的开源网页项目。  
目标是做一个 **以徐州为主题的美食共建地图**，让用户能看到值得去的店，点开了解详情，并一键跳转到常用地图 App 导航。

## 项目定位

这不是一个站内导航项目。  
它更像一个：
- 本地美食发现地图
- 城市风格化探索项目
- 可共建的徐州吃喝点位库

项目当前主路线已经确定为：
- **自定义徐州底图**
- **店铺点位覆盖**
- **店铺详情展示**
- **外跳地图 App 导航**

而不是依赖在线地图 API 做主地图。

## 本地启动

进入前端目录：

```bash
cd web
npm install
npm run dev
```

默认开发地址：

```bash
http://localhost:3000
```

---

## 环境变量

在 `web/` 下创建 `.env.local`：

```bash
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
NEXT_PUBLIC_AMAP_KEY=
```

说明：
- 当前项目主地图不依赖高德 JS API
- `NEXT_PUBLIC_AMAP_KEY` 仅作为可选增强能力保留
- 就算暂时没有高德 key，也不影响主流程开发

## 为什么选择自定义底图

因为这个项目的核心不是“站内导航”，而是：
- 让用户发现店
- 让用户知道店大概在哪
- 让用户点击后跳转到熟悉的地图 App

所以相比在线地图 API，**自定义底图更自由，也更符合项目气质**。
