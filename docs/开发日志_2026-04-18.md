### 3. 已修复的关键问题
#### 3.1 Next.js Turbopack 根目录识别错误
问题：在 OpenClaw workspace 多个 lockfile 共存时，Next 16 dev server 会错误识别项目根目录并 panic。  
修复：在 `web/next.config.ts` 中显式设置 `turbopack.root`。

#### 3.2 高德地图本地域名白名单限制
问题：高德 Web JS API 在本地开发时触发 `INVALID_USER_DOMAIN`，导致真实地图无法正常使用。  
结论：这不是当前 MVP 主路径，应下调为可选增强能力。

#### 3.3 自定义底图初版渲染空白
问题：GeoJSON 中 `MultiPolygon` 处理过于粗糙，SVG path 未正常闭合。  
修复：已改为按 `Polygon / MultiPolygon` 逐 ring 解析并生成 path。
